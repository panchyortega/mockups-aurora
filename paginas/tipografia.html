Documentación de Tipografías

1. Tipos de Texto

En esta sección, exploramos los diferentes tipos de texto que usamos en nuestro proyecto, incluyendo encabezados, párrafos, y texto de código. Cada tipo tiene un tamaño y estilo específico, diseñado para mejorar la legibilidad y accesibilidad.

2. Código de Estilos


/* Estilo de encabezado */
h1 {
  font-size: @size-text-h1;
  color: @color-font-h1;
}

/* Estilo de párrafo */
p {
  font-family: @font-body;
  font-size: @size-text-body-md;
}
