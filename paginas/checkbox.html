Componente Checkbox

El componente Checkbox permite a los usuarios seleccionar o deseleccionar opciones en una interfaz. Este componente es ideal para formularios y opciones donde el usuario puede elegir múltiples respuestas. Su apariencia es personalizable, y soporta estados como habilitado, deshabilitado, marcado y no marcado. Además, los checkboxes pueden agruparse en categorías para mejorar la organización y visualización de opciones.

Estilos (Less)

El estilo para el componente checkbox está dividido en tres partes principales: el contenedor del checkbox, el checkbox en sí y la etiqueta del checkbox. Aquí está el código de los estilos:

        }

          cursor: pointer;
        }
      }
      
      .checkbox {
