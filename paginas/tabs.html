Componente Nav-Tabs

El componente Nav-Tabs es una estructura que contiene varias pestañas (tabs) y permite alternar entre diferentes secciones de contenido dentro de una interfaz. Las pestañas son componentes modulares que se utilizan para organizar la información en la interfaz de usuario, mejorando la usabilidad y la experiencia del usuario.

El componente Nav-Tabs puede ser configurado para estar flotante (como un menú de navegación vertical) o fijo (para mostrar pestañas de contenido horizontal). Se puede adaptar según el tipo de interacción deseada.

Explicación del Código

El componente Nav-Tabs incluye varias instancias de Tabs, que son modulares y reutilizables. Cada tab representa una sección en el contenedor de navegación y puede incluir texto y/o íconos a la izquierda y derecha.


  {{#isFloating}}
  

    {{#tabs}}
      
    {{/tabs}}
  

  {{/isFloating}}
  
  {{#isFixed}}
  

    {{#tabs}}
      
    {{/tabs}}
  

  {{/isFixed}}

Estilos (Less)

Los estilos para las pestañas (tabs) varían según el estado y el tipo de tab (flotante o fijo). A continuación, mostramos cómo se manejan los diferentes estados y tipos de tabs en el código de los estilos:


  .nav-tabs {
    display: inline-flex;
    align-items: center;
    justify-content: flex-start;
    width: 100%;
  
    &.floating {
      padding: @padding-sm;
      gap: @gap-md;
      border-radius: @corner-radius-md;
      background-color: @col-surf-50;
    }
  
    &.fixed {
      padding: 0;
      gap: 0;
      background-color: transparent;
    }
  
    .tab {
      display: inline-flex;
      align-items: center;
      justify-content: center;
      gap: @gap-sm;
      padding: @padding-sm @padding-lg;
      font-family: @font-body;
      font-size: @size-text-body-md;
      font-weight: 600;
      border: none;
      cursor: pointer;
      transition: background-color 0.2s ease, color 0.2s ease, border-color 0.2s ease;
  
      &.default {
        background-color: @col-surf-50;
        color: @col-surf-700;
  
        &:hover {
          background-color: @col-surf-100;
        }
  
        &:active {
          background-color: @col-surf-200;
        }
      }
  
      &.selected {
        background-color: @col-prim-50;
        color: @col-prim-400;
  
        &:hover {
          background-color: @col-prim-100;
        }
  
        &:active {
          background-color: @col-prim-200;
        }
      }
  
      &.disabled {
        background-color: @col-surf-100;
        color: @col-surf-400;
        cursor: not-allowed;
      }
  
      .icon-left {
        margin-right: @gap-sm;
      }
  
      .icon-right {
        margin-left: @gap-sm;
      }
  
      .tab-text {
        font-size: @size-text-body-md;
        color: inherit;
      }
    }
  }
