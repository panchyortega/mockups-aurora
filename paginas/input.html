Componente Input

El componente Input permite a los usuarios ingresar texto en una interfaz. Es ideal para formularios donde los usuarios proporcionan información como nombres, direcciones o cualquier otro dato textual. El componente es flexible y personalizable con opciones como íconos, validación, y la capacidad de mostrar un texto de ayuda adicional.

Estilos (Less)

El estilo para el componente input está dividido en varias secciones, como el contenedor del checkbox, el checkbox mismo y la etiqueta del checkbox. Aquí está el código de los estilos:


      .input {
        display: flex;
        flex-direction: column;
        gap: @gap-xs;
        width: 100%;
        max-width: 419px;
        font-family: @font-body;
      
        &.fixed {
          width: 419px;
        }
      
        &.full {
          width: 100%;
        }
      
        /* Estilos del header (label, requirement, tooltip) */
        .input-header {
          display: flex;
          align-items: center;
          gap: @gap-xs;
          max-width: 100%;
          color: @col-surf-700;
      
          .input-label {
            font-family: @font-body;
            font-size: @size-text-body-md;
            color: @col-surf-700;
            line-height: @size-text-body-md * 1.6;
          }
      
          .input-requirement {
            font-family: @font-body;
            font-size: @size-text-body-md;
            color: @col-surf-700;
            line-height: @size-text-body-md * 1.6;
      
            /* Cambiar color en estado de error */
            .error & {
              color: @col-danger-200;
            }
          }
        }
      
        /* Estilos del input body (con íconos y caja de texto) */
        .input-body {
          display: flex;
          align-items: flex-start;

          .text-input {
            flex-grow: 1;
            color: @col-surf-500;

          }
      
          .icon-left, .icon-right {

          }
        }
      
        /* Estilo para texto de ayuda */
        .input-helptext {
