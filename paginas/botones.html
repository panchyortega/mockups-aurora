Componente: Botón

El componente de **Botón** es un elemento interactivo utilizado para realizar acciones dentro de la interfaz de usuario. Permite al usuario realizar interacciones como enviar formularios, navegar a otras páginas, o ejecutar funciones específicas de la plataforma. Los botones pueden tener diferentes estilos y variantes según su uso.

Explicación del Código

A continuación se presenta el código del componente de botón:


        .button {
          display: inline-flex;
          align-items: center;
          justify-content: center;
          gap: @gap-md;                  
          padding: @padding-sm @padding-lg;
          border-radius: @corner-radius-xs;
          font-family: @font-body;
          font-size: @size-text-body-md;
          font-weight: 600;
          line-height: @size-text-body-md * @lh-body;
          border-width: 2px;
          border-style: solid;
          cursor: pointer;
          margin-block-start: 0em;
          margin-block-end: 0em;
          margin-inline-start: 0px;
          margin-inline-end: 0px;
        
          // Variantes de color
          &.primary {
            background-color: @col-prim-400;
            color: @col-surf-50;
            border-color: transparent;
            &:hover {
              background-color: @col-prim-600;
            }
            &.disabled {
              background-color: @col-surf-200;
              color: @col-surf-500;
            }
          }
        }

Propiedades y Variantes del Botón

El componente de botón tiene varias variantes y propiedades que puedes configurar. A continuación, se muestra una tabla con las variantes de color y sus propiedades:

Tipo de Botón Descripción Estado de Hover Estado de Deshabilitado
Primary Botón principal de acción Cambia de color a un tono más oscuro Fondo gris, texto deshabilitado
Secondary Botón secundario con borde Fondo gris claro Fondo gris, texto deshabilitado
Danger Botón para acciones peligrosas Fondo rojo oscuro Fondo gris, texto deshabilitado
Success Botón para indicar éxito Fondo verde oscuro Fondo gris, texto deshabilitado
